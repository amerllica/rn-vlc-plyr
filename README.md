# rn-vlc-plyr

React Native VLC Player (Video/Audio Player) based on VLC

## Installation


```sh
npm install rn-vlc-plyr
```


## Usage


```js
import { RnVlcPlyrView } from "rn-vlc-plyr";

// ...

<RnVlcPlyrView color="tomato" />
```
